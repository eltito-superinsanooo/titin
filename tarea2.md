# **ejercicio**
## *Introducción*
### El Modelo de Objetos del Documento (DOM) es una estructura que usan los navegadores para representar páginas web como un conjunto de elementos organizados en forma de árbol. Esto permite a los desarrolladores interactuar con el contenido de manera dinámica.

![si](/titin/actividad2/img/foton.png)

## *Síntesis* 
### El DOM actúa como un puente entre HTML y JavaScript, permitiendo modificar elementos, estilos y contenido sin recargar la página. Cada parte del documento es un nodo que puede ser accedido y cambiado con métodos como getElementById o querySelector. Gracias a esto, se pueden crear páginas más interactivas y responder a acciones del usuario.
## *Reflexión* 
### El DOM es útil porque facilita la creación de sitios dinámicos. Permite mejorar la experiencia del usuario al hacer que las páginas reaccionen en tiempo real.
## *Conclusión* 
### En resumen, el DOM es clave para el desarrollo web moderno, ya que permite manipular y actualizar el contenido de forma rápida y eficiente.
